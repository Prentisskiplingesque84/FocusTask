## 🛠️ If something goes wrong

### The app will not open
- Check that the download finished
- Make sure you extracted the zip file if one was provided
- Try opening the file again
- Restart your PC if needed

### Notifications do not show
- Open Windows notification settings
- Allow alerts for FocusTask
- Make sure focus mode is off

### Export does not work
- Check that you have at least one task in the list
- Try exporting again
- Make sure the file is not open in another program

### Drag and drop does not move items
- Click and hold the subtask
- Drag it to the new position
- Release the mouse button only after the item is in place
